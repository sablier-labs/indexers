# Sablier Bob Subgraph

Bespoke subgraph for [Sablier Bob](https://docs.sablier.com/concepts/bob/overview), the price-gated token vaults
protocol. Bob lets users lock ERC-20 tokens in vaults that unlock when a target price is reached or a time period
expires.

This indexer is standalone — it is not part of the repo-wide type system and has no corresponding Envio indexer.
